Object verbergen - hidden

Object verbergen. Door middel van een display:none wordt het object verborgen.

Let op: Screen readers verbergen over het algemeen ook objecten met een display:none voor hun gebruikers. Als dit niet de bedoeling is en de content wel beschikbaar dient te zijn voor screenreader gebruikers is de helper class visually-hidden waarschijnlijk de betere optie.

Benodigde stappen:

  1. Voeg de class hidden toe aan het gewenste element.
