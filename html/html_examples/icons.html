Iconen

Voor het toevoegen van de Rijksoverheid-iconen binnen de website kan gebruikt gemaakt worden van een icoon-lettertype via de classes icon en icon-{"{naam}"}.

Standaard maken deze classes gebruik van het meegeleverde icoon-lettertype ro-icons-light. Dit lettertype bevat de iconen uit de Manon basis-set. Zie beschikbare iconen voor meer informatie over deze standaard iconen.

Daarnaast is het op twee manieren mogelijk om de iconen-set uit te breiden. Ten eerste kan je ervoor kiezen de uitgebreide icoon-set van dit thema te gebruiken. Ten tweede is het mogelijk om applicatie-specifieke iconen toe te voegen. Zie hiervoor de Manon-documentatie: andere iconen toevoegen

Iconen gebruiken

Voeg de class icon om gebruik te maken van de standaard icon-styling, en de class van het benodigde icoon, bijvoorbeeld: icon-home.

Paragraaf met icoon

Visueel voorbeeld:

HTML-voorbeeld:

                
<p class="icon icon-home">Lorem ipsum</p>

Uitgebreide icoon-set

Als de applicatie een icoon nodig heeft, dat niet in de standaard-set zit, dan is het mogelijk om de uitgebreide icoon-set te gebruiken. Deze bevat een grotere sub-set van de rijkshuisstijl-iconen.

Importeer hiervoor na het thema ook de uitgebreide icoon-set:

/* scss/main.scss */
@use "manon";
@use "@minvws/nl-rdo-rijksoverheid-ui-theme";
@use "@minvws/nl-rdo-rijksoverheid-ui-theme/components";
// voeg deze regel toe:
@use "@minvws/nl-rdo-rijksoverheid-ui-theme/icons-extended";
