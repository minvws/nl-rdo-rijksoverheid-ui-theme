Iconen

Voor het toevoegen van de Rijksoverheid-iconen binnen de website kan gebruikt gemaakt worden van de unicode-codes.

Overzicht van de beschikbare iconen:

  • Basisset
  • Gezondheid en zorg
  • Computer en internet
  • Gebouwen

Iconen gebruiken

Voeg de class icon om gebruik te maken van de standaard icon-styling, en de class van het benodigde icoon, bijvoorbeeld: icon-home.

Paragraaf met icoon

Visueel voorbeeld:

Lorem ipsum

Html-voorbeeld:

                
<p class="icon icon-home">Lorem ipsum</p>

Knop met icoon en tekst

Visueel voorbeeld:

Hoofdpagina

Html-voorbeeld:

                
<a href="../index.html" class="button"><span class=" icon icon-home"></span>Hoofdpagina</a>

Icoon toevoegen via css

Voor het toevoegen of overschrijven van iconen kunnen deze ook via een eigen stylesheet toegevoegd worden. Gebruik hiervoor de onderstaande css.

Aandachtspunten:

  • Plaats het icoon via een pseudo-element Bijvoorbeeld: :before of :after op het gewenste element.
  • Voeg de bijbehorende icoon-code toe binnen het content attribuut. Voor een overzicht van de beschikbare iconen zie: iconen

CSS-voorbeeld before:

                
.example-object:before {
  content: "\E805";
  font-family: "RO Icons";
  font-size: 1.125rem;
  color: inherit;
  position: static;
  display: inline-flex;
  justify-content: center;
  align-items: center;
  margin-right: 0.5rem;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale; /* Firefox */
  font-smoothing: antialiased;
}

CSS-voorbeeld after:

                
.example-object:after {
  content: "\E805";
  font-family: "RO Icons";
  font-size: 1.125rem;
  color: inherit;
  position: static;
  display: inline-flex;
  justify-content: center;
  align-items: center;
  margin-left: 0.5rem;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale; /* Firefox */
  font-smoothing: antialiased;
}
