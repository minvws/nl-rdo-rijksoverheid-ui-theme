Kleurcontrast testpagina

Op deze pagina staan alle verschillende kleurcombinaties om contrast te kunnen controleren.

  • "WCAG AA"-standaard vraagt om een kleurcontrast van 4.5:1 op normale tekst en 3:1 op grote op grote tekst. Voor meer informatie zie: WCAG AA eisen
  • "WCAG AA"-standaard vraagt om een kleurcontrast van 7:1 op normale tekst en 4.5:1 op grote op grote tekst. Voor meer informatie zie: "WCAG AAA"-eisen
  • De standaard aangeboden kleurenset binnen dit framework voldoent aan de "WCAG AA"-eisen. Een "WCAG AAA" kleurenset kan optioneel aangezet worden in het ro-branding.scss binnen de ontwikkelaarversie.
